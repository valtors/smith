/ 01 / CORE CAPABILITIES

MCP without the manual labor

install, compose, secure, and switch MCP server profiles. no json editing, no manual restarts, no copy-paste errors.

01.0CORE

one command install

smith parses the package, writes the config, verifies the server starts. no json editing, no manual restarts, no copy-paste errors.

$ smith install filesystem
installed: filesystem (12 tools)
02.0CORE

compose servers

merge multiple MCP servers into one endpoint. one connection, many tools. automatic json-rpc multiplexing under the hood.

$ smith compose fs git mem
merged: 23 tools on :9847
03.0SEC

security analysis

static analysis on every server before it runs. detects dangerous patterns, env leaks, unsafe file access. catch problems before they execute.

$ smith scan
warn: env access detected
blocked: shell exec
