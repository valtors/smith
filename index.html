01 features

six tools. one binary.

focused commands for the entire MCP server lifecycle. install, compose, secure, profile, search, remove.

+

install

one command. smith parses the package, writes the config, verifies the server starts. no json editing.

=

compose

merge multiple MCP servers into one endpoint. one connection, many tools. automatic json-rpc multiplexing.

!

secure

static analysis on every server before it runs. detects dangerous patterns, env leaks, unsafe file access.

@

profiles

separate configs for work, home, production. switch profiles with one command. no symlink hacks.

*

registry

github-based package index. no separate server. search and discover servers from the community.

-

remove

clean uninstall. removes config entries, stops processes, frees ports. no orphaned servers.

02 vs manual config

stop editing json. start shipping.

every MCP server install is the same ritual: open config, paste json, set paths, restart, hope. smith does it in one command.

manual config

copy a json blob. paste it into a config file. restart everything. hope it works.

  • edit json by hand every time
  • no security checks before running
  • one server per connection
  • profiles via symlink hacks
  • no search, no discovery

smith

one command. installed, verified, ready. compose multiple servers into one endpoint.

  • one command install and remove
  • static security analysis built in
  • compose merges servers into one
  • first-class profile switching
  • github-based registry search
